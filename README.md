# 🐧 Penguin Fashion – E-commerce Landing Page

Penguin Fashion is a modern and responsive **e-commerce landing page** built using **HTML and Tailwind CSS**.  
The website showcases winter fashion products with a clean UI, product cards, and a responsive layout suitable for mobile, tablet, and desktop devices.

## ✨ Features

- 📱 Fully responsive design
- 🧥 Product showcase sections
- 🎨 Clean and modern UI
- ⚡ Built with Tailwind CSS utility classes
- 🛒 Product cards with price and buy button
- 🧭 Simple navigation bar
- 📦 Facilities / service section
- 🖼️ Optimized product images
